For a mid-term project at Grand Circus' Java Bootcamp, my group was tasked with creating a library terminal that allows people to see a list of books, an adaptive list of book genres, check out, return and donate books to a library.

This program was able to take input from any case and properly format it to the proper capitalization.
When a user was donating a book, the program is also able to adapt its list of genres to include any newly added genres and allow the user to input multiple genres for a single book.
